# Movie Recommendation System using TMDB Dataset

# Description
This project is a content-based movie recommendation system built using Python and Machine Learning. It recommends movies similar to a selected movie by analyzing textual features such as genres, overview, keywords, cast, and crew.
The system uses Count Vectorization and Cosine Similarity to find and suggest the most relevant movies.
